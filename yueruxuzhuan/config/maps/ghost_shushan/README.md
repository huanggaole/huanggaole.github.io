## 地图描述

阴森恐怖的鬼界蜀山，与人界蜀山截然不同的深邃之地。这里是林月如在鬼界的第一站，充满了神秘的鬼魂和古老的传说。

## 剧情作用

1. **过渡地点**: 从鬼门关到鬼界苏州的中转站
2. **信息收集**: 通过与NPC对话了解鬼界规则和背景
3. **氛围营造**: 展现鬼界的神秘和阴森氛围
4. **角色发展**: 林月如适应鬼界环境的重要阶段

## 设计理念

- **视觉风格**: 深紫色主调，营造神秘鬼界氛围
- **交互设计**: 多个NPC提供丰富的对话内容
- **动画效果**: 脉动动画增强神秘感
- **空间布局**: 合理分布NPC和装饰元素，避免拥挤
